#### Update a plan

Updates the name of a plan. Other plan details (price, interval, etc.) are, by design, not editable.

##### Usage

```php
$plan = $stripe->plans()->update('monthly', [
    'name' => 'Monthly Subscription',
]);

echo $plan['name'];
```
